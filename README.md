# Proyecto mood tracker

## Disposición del repositorio

https://srchicken6711.github.io/Mood_tracker_0.1/

### Assets
Dentro de esta carpeta se guardaran imagenes o recursos que necesite la página dentro del directorio img/

### css
Dentro de esta carpeta se encuentra el archivo styles.css que servira para darle diseño a la página

### .gitignore
Archivos que no deberian subirse al repositorio global, en caso de no tenerlo aun crea un archivo con el nombre .gitignore y dentro de este pega lo siguiente

~~~
# Database
*.db
*.sqlite
*.sqlite3
database/data.db
database/data.sqlite

# OS
.DS_Store
Thumbs.db

# Editor
.vscode/
.idea/

# Environment variables
.env
.env.local

# Logs
*.log

# Node (si usas npm)
node_modules/
package-lock.json
~~~

### index.html
Archivo principal de la página web
